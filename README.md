# CouponDonation - Transforming Giving Through Generosity

CouponDonation connects generous donors with those in need. Transform lives through the power of giving with our coupon donation platform.

## About

CouponDonation is a platform that enables donors to contribute through partner brand coupons, directly supporting recipients in need. Our mission is to make giving accessible, transparent, and impactful.

## Technology Stack

- React 18 with TypeScript
- Vite for development and building
- Tailwind CSS for styling
- shadcn/ui component library
- Supabase for backend (auth, database, edge functions)
- Stripe for payment processing

## Getting Started

### Prerequisites
- Node.js 18+ (install with [nvm](https://github.com/nvm-sh/nvm))
- npm or bun

### Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

## Features

- **Donate**: Support causes you care about through partner brand coupons
- **Apply for Help**: Recipients can apply for assistance
